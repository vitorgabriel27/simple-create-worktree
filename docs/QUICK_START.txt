WORKTREE SCRIPTS - QUICK START
==============================

1) Install:
   ./install.sh

2) Ensure PATH contains ~/.local/bin:
   export PATH="$HOME/.local/bin:$PATH"

3) Create worktree:
   git-create-worktree feat/my-feature

4) Create new branch + worktree:
   git-create-worktree -b feat/new-feature develop

5) Remove worktree:
   git-remove-worktree feat/my-feature

6) Update current branch in the current worktree:
   git-update-worktree

7) Update a specific local branch:
   git-update-worktree develop

8) Merge remote branch into local branch (in active worktree):
   git-merge-worktree feat/chart feat/dashboard

9) Merge local branch into local branch (in active worktree):
   git-merge-worktree -l feat/chart feat/dashboard
